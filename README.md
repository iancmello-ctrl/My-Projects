# 🎮 Servidor de Jogos + Node.js + Telegram

🚀 Projeto de monitoramento automático de servidor de jogos usando **Node.js** e integração com **Telegram Bot API**.

## ✨ Funcionalidades
- 📊 Logs coloridos com [chalk](https://www.npmjs.com/package/chalk)
- 🔄 Reinício automático em caso de falha
- 📱 Alertas em tempo real via Telegram (usando [Axios](https://www.npmjs.com/package/axios))
- 🛡️ Proteção com Helmet e Rate Limit
- ✅ Validação de entradas com Validator

## ⚙️ Como rodar
```bash
git clone https://github.com/seuusuario/Servidor-Jogos-Node-Telegram
cd Servidor-Jogos-Node-Telegram
npm install
npm start

Instale essas dependências no seu Terminal: npm install express express-rate-limit helmet validator chalk axios
